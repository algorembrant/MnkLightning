# MnkLightning

MonkeyLighting is a cheat/hack for those who whish to manipulate their typing speed on [Monkeytype.com](https://monkeytype.com/) at the same time not getting banned. 

This project repository is for fun only and serve as validation for the 'script_candidate.js' wheter it worked or not, a cheat for mimicking human imperfection typing-behavior at the same time achieving a high typing speed (wpm).

# How to use?

1. Open [Monkeytype.com](https://monkeytype.com/).<br>
2. Open the browser's developer tools (usually by pressing F12 or right-clicking on the page and selecting "Inspect" or "Inspect Element").<br>
3. Go to the "Console" tab. (ctrl+shift+j for windows)<br>
4. Copy and paste the code from the `script_candidate.js` file into the console.<br>
5. Press Enter to run the code.<br>
6. The cheat should now be active and will begin typing automatically when you start typing.<br>
